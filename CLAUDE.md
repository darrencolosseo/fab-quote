## What This App Is

**Fab Quote** — internal quoting tool for **Fab Garage Doors** (Australian garage door installer).

**Purpose:**
- Replace physical notebooks + manual calculations
- Auto-calculate board count, frame costs, cladding, margin, totals live
- Record-keep all past jobs with full breakdown
- Generate and download a branded PDF quote named `Customer Name - Address Suburb.pdf`

**Stack:** React + Vite (v8) · Tailwind CSS · Supabase (DB + auth) · jsPDF (PDF) · React Router

**Dev server:** `npm run dev` in `/Users/darren/Desktop/Example Website/fab-quote/`

## Core Pricing Logic (`src/config/pricing.js`)

### Board Count Formula

From Excel Sheet4:
```
boards = CEIL(width_mm / boardWidth_mm)
if height_mm > 0 AND height_mm < 2900:
    boards = CEIL(boards / 2)   ← height cut optimisation (2 cuts per board)
```

- Default board width: **150mm** (Castellated Board, WPC, etc.)
- Default board cost: **$180/board**
- Height threshold: **2900mm** — below this, boards are cut in half lengthways (halves needed)

### Margin

- **30%** applied to all sections: `total = subtotal * 1.30`
- `MARGIN_PCT = 0.30`

### Frame / Motor Package — Auto-Estimate Table

Sourced from Excel `MASTER GARAGE PRICE LIST 2026.xlsx`:

## Key UX Decisions

- **Frame cost auto-fills** when width + door type are selected. User can tap "Override" to enter manually, then "← Restore auto" to revert.
- **Size presets** (width chips: 2400/2700/3000/4800/5000/5500/6100, height chips: 2100/2400/2700) for fast tap entry.
- **Board count shows green "cut optimised ✓"** when height < 2900mm.
- **All totals recalculate live** via `useMemo` — no submit button needed to see results.
- **Grand total flashes orange briefly** via `useFlash` hook when value changes.
- **Toast notifications** replace all `alert()` calls. Module-level: `showToast(msg, 'success'|'error')`.
- **On save:** PDF auto-downloads named `Customer Name - Address Suburb.pdf`, then navigates to job detail.
- **EditQuote** only edits customer details, notes, and an optional price override. To change door specs, create a new quote.
- **Status tabs** on JobList: All / Active / Won / Lost. Stats row shows total quotes, pipeline $, won revenue, conversion %.

## How to Update Prices

1. Open `src/config/pricing.js`
2. Update the relevant constant or table:
   - **Board cost:** `BOARD_COST`
   - **Frame costs:** `FRAME_BRACKETS` array
   - **Front door parts:** `FRONT_DOOR_DEFAULTS`
   - **Wall defaults:** `WALL_DEFAULTS`
3. Save — all calculations update automatically throughout the app.
